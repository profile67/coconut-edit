# Coconut Edit — AI Video Studio

A clean, modern, fully interactive **dark-mode video editor** built as a single HTML file.

### Features
- Persistent login (stays logged in until you log out)
- Live clock + user email + custom profile picture
- Drag & Drop or select video
- Real video preview with native controls
- Editing tools: Trim, Zoom (Ken Burns), Brightness, Contrast, Grayscale, Speed control
- One-click Export & Download
- Beautiful Apple-inspired dark UI with red accents

### How to Use
1. Download `index.html`
2. Open it in any modern browser (Chrome recommended)
3. Sign up with any email + password
4. Drag or select a video and start editing

Made with ❤️ for fast creators.

---

**Note**: This is a demo built in pure HTML/CSS/JS. No backend, no real rendering engine (effects are applied live on preview).
